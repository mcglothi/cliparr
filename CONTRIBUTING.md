# Contributing

1. Open an issue describing the bug/feature.
2. Fork, branch, and submit a PR.
3. Add tests and docs for behavior changes.
4. Never commit credentials or personal account data.

## Development

```bash
docker compose -f ops/compose/docker-compose.yml up -d --build
```
